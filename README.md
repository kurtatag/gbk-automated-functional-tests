## EKS Automated Functional Tests

To run the example you need to install [maven (3.1.1 or above)][maven] and [git][git]. 
To generate Allure Report you should perform following steps:

```bash
$ git clone ...
$ ./mvnw clean test
$ ./mvnw allure:report
```
